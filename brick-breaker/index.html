How to Play:

Objective: Break all bricks with the ball without letting it fall

Controls:

  • Move Mouse or Arrow Keys ← → to control paddle
  • Space: Launch ball (at start)
  • P: Pause game

Levels:

  • Easy: Slow ball speed, 3 lives
  • Medium: Normal speed, 3 lives
  • Hard: Fast ball, 2 lives
  • Extreme: Multiple balls, 3 lives

Tips: Aim for the sides of the paddle for angled shots!
