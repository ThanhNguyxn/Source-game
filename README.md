# 🎮 Mini Games Collection

A collection of classic games built with HTML, CSS, and JavaScript. Play these games directly in your browser - no installation required!

## 🎯 Games Included

### 1. 🐍 Snake Game
Classic snake game where you control a snake to eat food and grow longer. Don't hit the walls or yourself!

**Features:**
- Smooth controls with arrow keys or WASD
- Score tracking with local high score
- Pause and resume functionality
- Responsive design

[Play Snake Game](./snake-game/index.html)

---

### 2. ❌⭕ Tic Tac Toe
The timeless two-player game. Take turns marking X's and O's on a 3x3 grid.

**Features:**
- Two-player local gameplay
- Score tracking for both players
- Win detection with visual feedback
- Persistent score history

[Play Tic Tac Toe](./tic-tac-toe/index.html)

---

### 3. 🎲 2048 Game
Slide numbered tiles on a grid to combine them and create a tile with the number 2048!

**Features:**
- Smooth tile animations
- Score and best score tracking
- Win/lose detection
- Arrow key controls

[Play 2048](./2048-game/index.html)

---

### 4. 🐦 Flappy Bird
Navigate a bird through pipes by tapping or pressing space. How far can you go?

**Features:**
- Simple one-button controls
- Endless gameplay
- High score tracking
- Smooth animations

[Play Flappy Bird](./flappy-bird/index.html)

---

### 5. 🧠 Memory Card Game
Test your memory by matching pairs of emoji cards!

**Features:**
- 16 cards (8 pairs) to match
- Move counter
- Timer with best time tracking
- Smooth flip animations

[Play Memory Game](./memory-card-game/index.html)

---

## 🚀 How to Play

1. **Clone this repository:**
   ```bash
   git clone https://github.com/yourusername/mini-games-collection.git
   ```

2. **Open any game:**
   - Navigate to the game folder
   - Open `index.html` in your web browser
   - Start playing!

Alternatively, you can play directly by opening the HTML files in your browser.

## 🎮 Game Controls

### Snake Game
- Arrow Keys / WASD - Move the snake
- Start Button - Begin game
- Pause Button - Pause/Resume

### Tic Tac Toe
- Mouse Click - Place your mark

### 2048
- Arrow Keys - Slide tiles

### Flappy Bird
- Space / Mouse Click - Flap

### Memory Card Game
- Mouse Click - Flip cards
