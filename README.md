## 🎯 Complete Game Collection

### 🕹️ Arcade Classics (7 Games)

#### 1. 🐍 Snake Game
Classic arcade game where you control a growing snake.

**Features:**
- Smooth snake movement with WASD or Arrow Keys
- Food spawning system
- Self-collision detection
- High score tracking with localStorage
- Pause/Resume functionality
- Speed increases as snake grows

**Controls:** Arrow Keys / WASD  
**Objective:** Eat food to grow longer without hitting walls or yourself

---

#### 2. 🎯 Breakout
Break all the colored bricks with your paddle and ball.

**Features:**
- Multiple brick rows with different colors
- Paddle control via Arrow Keys or Mouse
- Ball physics with angle calculation
- Lives system (3 lives)
- Progressive level difficulty
- Score multiplier

**Controls:** Arrow Keys / Mouse  
**Objective:** Break all bricks without dropping the ball

---

#### 3. 🎮 Tetris
The legendary falling blocks puzzle game.

**Features:**
- 7 classic tetromino shapes
- Next piece preview system
- Line clearing with animations
- Progressive speed increase
- Hold piece functionality (coming soon)
- Ghost piece preview (coming soon)

**Controls:**  
- ← → : Move piece
- ↑ : Rotate piece
- ↓ : Soft drop
- Space : Hard drop

**Objective:** Clear lines by filling rows completely

---

#### 4. 🏓 Pong
The first video game ever created - table tennis simulation.

**Features:**
- AI opponent with adaptive difficulty
- Smooth paddle physics
- Ball speed increases over time
- First to 5 points wins
- Score tracking

**Controls:** Up/Down Arrow Keys or W/S  
**Objective:** Beat the computer by getting to 5 points first

---

#### 5. 👾 Space Invaders
Defend Earth from waves of alien invaders!

**Features:**
- Classic alien movement patterns
- Player shooting mechanics
- Alien return fire
- Progressive difficulty
- Lives system
- High score tracking

**Controls:**  
- ← → : Move ship
- Space : Shoot

**Objective:** Destroy all aliens before they reach you

---

#### 6. 🐦 Flappy Bird
Navigate through pipes in this challenging arcade game.

**Features:**
- One-button control system
- Procedurally generated pipes
- Smooth physics engine
- Endless gameplay
- High score tracking
- Death screen with score

**Controls:** Space / Mouse Click  
**Objective:** Fly through as many pipes as possible

---

#### 7. 🦖 Dino Run
Chrome's famous offline dinosaur game!

**Features:**
- Jump and duck mechanics
- Obstacle variety (cacti and flying birds)
- Speed increases over time
- Score based on distance
- Day/night cycle (coming soon)

**Controls:**  
- Space/↑ : Jump
- ↓ : Duck

**Objective:** Survive as long as possible

---

### 🧩 Puzzle & Strategy (6 Games)

#### 8. 🎲 2048
Slide numbered tiles to create the 2048 tile.

**Features:**
- 4x4 grid gameplay
- Tile merging animations
- Smooth sliding physics
- Undo move (coming soon)
- Best score tracking
- Win/lose detection

**Controls:** Arrow Keys  
**Objective:** Combine tiles to reach 2048

---

#### 9. 💣 Minesweeper
Classic mine detection puzzle.

**Features:**
- **3 difficulty levels:**
  - Easy: 8x8 grid, 10 mines
  - Medium: 16x16 grid, 40 mines
  - Hard: 16x30 grid, 99 mines
- First-click safety (never hits mine)
- Flag system (right-click)
- Timer and move counter
- Best time tracking

**Controls:**  
- Left Click: Reveal cell
- Right Click: Place/remove flag

**Objective:** Find all mines without clicking on them

---

#### 10. 🔢 Sudoku
Fill the 9x9 grid with numbers 1-9.

**Features:**
- **3 difficulty levels:**
  - Easy: 40 empty cells
  - Medium: 50 empty cells
  - Hard: 60 empty cells
- Auto-generated valid puzzles
- Hint system
- Error checking
- Timer
- Number pad interface
- Mistake counter (max 3)

**Controls:** Click cell, then number  
**Objective:** Fill grid following Sudoku rules

**Rules:**
- Each row must contain 1-9
- Each column must contain 1-9
- Each 3x3 box must contain 1-9

---

#### 11. ❌⭕ Tic Tac Toe
The timeless two-player strategy game.

**Features:**
- Local multiplayer
- Win detection with animation
- Score tracking for both players
- Draw detection
- Reset score option
- Persistent score history

**Controls:** Mouse Click  
**Objective:** Get three in a row (horizontal, vertical, or diagonal)

---

#### 12. 🔴🟡 Connect Four
Connect four discs vertically, horizontally, or diagonally.

**Features:**
- Two-player local gameplay
- Gravity physics for dropping discs
- Win detection with highlighting
- Score tracking
- Smooth animations
- Reset score option

**Controls:** Click on column  
**Objective:** Connect four of your discs in a row

---

#### 13. 🧩 Sliding Puzzle
Classic 15-tile number sliding puzzle.

**Features:**
- 4x4 grid with numbers 1-15
- Move counter
- Timer
- Best moves tracking
- Shuffle function
- Show solution button
- Visual feedback for correct tiles

**Controls:** Click tile adjacent to empty space  
**Objective:** Arrange tiles in order 1-15

Test your knowledge across multiple categories.

  - Easy: 10 points

### 🎴 Card & Action (4 Games)

#### 21. 🃏 Blackjack
Classic card game with betting system.

**Features:**
- Full 52-card deck
- Betting system (chips start at 1000)
- Hit, Stand, Double Down options
- Dealer AI (hits until 17)
- Card counting display
- Win/lose/push detection
- Animated card dealing
- Multiple rounds
- Bankruptcy detection

**Controls:** Click buttons  
**Rules:**
- Get closer to 21 than dealer
- Ace = 1 or 11
- Face cards = 10
- Bust if over 21

- Level system
